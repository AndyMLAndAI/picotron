FAQ

Can I run Picotron on a T4?

That is a target environment for the project. The hardware policy selects fp16 on Turing/T4 and rejects bf16 there. Run a small Kaggle smoke test before a long job; performance and memory claims are configuration-specific.

Does the site start a run or send my config anywhere?

No. The config builder is client-side JavaScript only. It produces a YAML preview and copy button; it does not upload data, contact the Hub, or execute commands.

Can I use an existing Hugging Face causal LM?

Yes for the SFT/DPO adapters when the model follows the normal causal-LM logits contract. The native pretraining CLI constructs Picotron's own decoder model from its strict configuration.

What happens when an optional backend fails?

Optional acceleration is guarded. Picotron reports the issue and returns to the relevant PyTorch path rather than silently continuing on an unknown implementation.
